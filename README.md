# INT Hierarchy Analysis in Autism Spectrum Disorder

MATLAB scripts accompanying:

**Shikauchi Y., Aoki R., Itahashi T., Shimizu M., Naoe T., Okimura T, Ohta H, Hashimoto R., and Nakamura M.**  
*Preserved Intrinsic Neural Timescale Organization with Hierarchical Variation in Autism Spectrum Disorder*  
bioRxiv (2026)  
DOI: 10.64898/2026.02.27.708484

Preprint:  
https://www.biorxiv.org/content/10.64898/2026.02.27.708484v2

## Contents
This repository contains MATLAB scripts used for the analyses developed in this study, including:

- Template-based modeling of individual INT profiles
- Estimation of global offset, hierarchical scaling, and residual RMS
- Spin-based spatial null testing for sensory hierarchy analyses

## Data Availability

Raw and preprocessed imaging data were obtained from the Brain/MINDS Beyond Human Brain MRI Project and are available upon request through the official data-sharing portal:

https://hbm.brainminds-beyond.jp/data.html

## Notes

MRI preprocessing pipelines and INT estimation procedures followed previously published methodologies and are therefore not included in this repository. This repository contains only the analysis scripts developed for the present study.
